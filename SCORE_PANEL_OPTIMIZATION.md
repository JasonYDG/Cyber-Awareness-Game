## 🚀 技术实现

### 颜色分类算法
```javascript
// 根据分数自动分类颜色
if (threat.points <= 15) {
    color = '#00ff00'; // 绿色 - 10分档
} else if (threat.points <= 25) {
    color = '#ffaa00'; // 橙色 - 20分档
} else {
    color = '#ff0040'; // 红色 - 30分档
}
```

### 分数标准化
```javascript
// 简化分数为3个档次
if (threat.points <= 15) {
    threat.points = 10;
} else if (threat.points <= 25) {
    threat.points = 20;
} else {
    threat.points = 30;
}
```
